# Lofty Skills Repository

This repository contains Claude Code Skills and MCP Server configurations for the Lofty team.

## Installation

### Install a skill directly:

```bash
claude skill add <skill-name>
```

### Or download and install:

```bash
curl -LO https://github.com/kratos06/lofty-skills/raw/main/skill-files/<skill-name>.skill
claude skill add <skill-name>.skill
```
