# Filesystem Integration

Local filesystem navigation, search, and file operations (Official)

## Configuration

## Commands

### List directory

```bash
ls -la /path/to/dir
```

### Read file

```bash
cat /path/to/file
```

### Write file

```bash
echo "content" > /path/to/file
```

### Create directory

```bash
mkdir -p /path/to/new/dir
```

### Find files

```bash
find /path -name "*.txt"
```
