# devops

DevOps skill for CI/CD, infrastructure, and deployment automation

## Installation

```bash
claude skill add devops
```

Or download the .skill file:

```bash
curl -LO https://github.com/kratos06/lofty-skills/raw/main/skill-files/devops.skill
claude skill add devops.skill
```

## Quick Start

After installation, the devops skill will be available in Claude Code.
