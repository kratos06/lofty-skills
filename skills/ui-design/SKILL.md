# ui-design

UI design system toolkit

## Installation

```bash
claude skill add ui-design
```

Or download the .skill file:

```bash
curl -LO https://github.com/kratos06/lofty-skills/raw/main/skill-files/ui-design.skill
claude skill add ui-design.skill
```

## Quick Start

After installation, the ui-design skill will be available in Claude Code.
