# code-review-skill

Deep code review for architecture, security, and quality

## Installation

```bash
claude skill add code-review-skill
```

Or download the .skill file:

```bash
curl -LO https://github.com/kratos06/lofty-skills/raw/main/skill-files/code-review-skill.skill
claude skill add code-review-skill.skill
```

## Quick Start

After installation, the code-review-skill skill will be available in Claude Code.
