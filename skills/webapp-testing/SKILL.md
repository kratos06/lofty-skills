# webapp-testing

Test local web applications using Playwright for UI verification and end-to-end testing

## Installation

```bash
claude skill add webapp-testing
```

Or download the .skill file:

```bash
curl -LO https://github.com/kratos06/lofty-skills/raw/main/skill-files/webapp-testing.skill
claude skill add webapp-testing.skill
```

## Quick Start

After installation, the webapp-testing skill will be available in Claude Code.
