# ml-engineer

ML engineering for model deployment and MLOps

## Installation

```bash
claude skill add ml-engineer
```

Or download the .skill file:

```bash
curl -LO https://github.com/kratos06/lofty-skills/raw/main/skill-files/ml-engineer.skill
claude skill add ml-engineer.skill
```

## Quick Start

After installation, the ml-engineer skill will be available in Claude Code.
