# skill-creator

Interactive skill creation tool that guides you through building new Claude skills

## Installation

```bash
claude skill add skill-creator
```

Or download the .skill file:

```bash
curl -LO https://github.com/kratos06/lofty-skills/raw/main/skill-files/skill-creator.skill
claude skill add skill-creator.skill
```

## Quick Start

After installation, the skill-creator skill will be available in Claude Code.
