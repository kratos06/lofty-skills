# planning

Implementation planning for multi-step tasks

## Installation

```bash
claude skill add planning
```

Or download the .skill file:

```bash
curl -LO https://github.com/kratos06/lofty-skills/raw/main/skill-files/planning.skill
claude skill add planning.skill
```

## Quick Start

After installation, the planning skill will be available in Claude Code.
