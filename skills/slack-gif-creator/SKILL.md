# slack-gif-creator

Create animated GIFs optimized for Slack's size constraints

## Installation

```bash
claude skill add slack-gif-creator
```

Or download the .skill file:

```bash
curl -LO https://github.com/kratos06/lofty-skills/raw/main/skill-files/slack-gif-creator.skill
claude skill add slack-gif-creator.skill
```

## Quick Start

After installation, the slack-gif-creator skill will be available in Claude Code.
