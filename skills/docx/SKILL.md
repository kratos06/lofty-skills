# docx

Create, edit, and analyze Word documents with support for tracked changes, templates, and formatting

## Installation

```bash
claude skill add docx
```

Or download the .skill file:

```bash
curl -LO https://github.com/kratos06/lofty-skills/raw/main/skill-files/docx.skill
claude skill add docx.skill
```

## Quick Start

After installation, the docx skill will be available in Claude Code.
