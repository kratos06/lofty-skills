## Commands

### List Cloud Storage buckets

```bash
gcloud storage ls
```

### List VM instances

```bash
gcloud compute instances list
```

### List Cloud Functions

```bash
gcloud functions list
```

### List Cloud Run services

```bash
gcloud run services list
```

### Query BigQuery

```bash
bq query "SELECT * FROM dataset.table LIMIT 10"
```
