# data-scientist

Data science with ML, statistics, and analytics

## Installation

```bash
claude skill add data-scientist
```

Or download the .skill file:

```bash
curl -LO https://github.com/kratos06/lofty-skills/raw/main/skill-files/data-scientist.skill
claude skill add data-scientist.skill
```

## Quick Start

After installation, the data-scientist skill will be available in Claude Code.
