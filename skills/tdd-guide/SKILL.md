# tdd-guide

Test-driven development guide

## Installation

```bash
claude skill add tdd-guide
```

Or download the .skill file:

```bash
curl -LO https://github.com/kratos06/lofty-skills/raw/main/skill-files/tdd-guide.skill
claude skill add tdd-guide.skill
```

## Quick Start

After installation, the tdd-guide skill will be available in Claude Code.
