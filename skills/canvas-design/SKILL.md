# canvas-design

Create beautiful visual art in PNG and PDF formats using design principles

## Installation

```bash
claude skill add canvas-design
```

Or download the .skill file:

```bash
curl -LO https://github.com/kratos06/lofty-skills/raw/main/skill-files/canvas-design.skill
claude skill add canvas-design.skill
```

## Quick Start

After installation, the canvas-design skill will be available in Claude Code.
