# senior-frontend

Senior frontend development with React, Next.js, TypeScript, and Tailwind

## Installation

```bash
claude skill add senior-frontend
```

Or download the .skill file:

```bash
curl -LO https://github.com/kratos06/lofty-skills/raw/main/skill-files/senior-frontend.skill
claude skill add senior-frontend.skill
```

## Quick Start

After installation, the senior-frontend skill will be available in Claude Code.
